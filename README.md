# PM2.5-Sensor
Arduino code for sensors

This is an Arduino code for a PM2.5 sensor based on Linkit 7697.
Sensor: PMS5003GT

Functions:
1. Connect to WIFI
2. Connect to server
3. Turn on sensor
4. Send data every 45 seconds
5. If disconnected from wifi or server, it will turn off sensor and try to reconnect
